# Mock LeRobot v3 Good Dataset

This is a mock/tutorial LeRobot-style dataset used only for `lerobot-quality-gates` tests.

It documents the robot embodiment, sensors, task family, and limitations. The mock robot is a tabletop arm with one front camera sensor, a 6D action space, and a 6D state vector. The task is synthetic block placement. Limitations: the files are tiny placeholders and must not be treated as real robotics data.

The dataset includes human intervention and recovery labels for review workflow testing.
