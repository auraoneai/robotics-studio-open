# Quickstart

1. Run `pnpm dev` and open `http://127.0.0.1:5173`.
2. Choose `Explore sample` for the 96-record synthetic fixture, or choose
   `Open my dataset` for JSON or JSONL episode metadata.
3. Confirm the provenance banner and unknown-field coverage.
4. Use Browse to filter and select an episode.
5. Use Review to inspect one active visual sensor, health metadata, language,
   transport, timeline, and the episode decision.
6. Run deterministic Failure Intelligence, Sensor QA, or one to four mock VLA
   trials.
7. Choose Export, inspect Source, Destination, Scope, and Artifacts, then
   confirm a local ZIP download.

Binary files are not parsed by this source build. Convert their episode
metadata to JSON or JSONL before import.
