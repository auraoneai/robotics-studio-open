# Keyboard Shortcuts

| Shortcut | Action |
| --- | --- |
| Cmd/Ctrl+O | Open JSON or JSONL metadata |
| Cmd/Ctrl+K | Open or close the command palette |
| Cmd/Ctrl+P | Switch to the next open dataset |
| Cmd/Ctrl+Shift+V | Save the current local view |
| Cmd/Ctrl+Shift+E | Open local export |
| Space | Toggle playback when focus is not in a control |
| Left / Right | Move one frame |
| Shift+Left / Shift+Right | Move ten frames |
| J / K | Next / previous episode |
| T | Apply the local second-pass taxonomy tag |
| B | Add a review phase boundary when duration is known |
| R | Toggle reviewed state |
| S / F | Mark success / failure |
| C | Open Failure Intelligence |
| V | Open the deterministic VLA probe |
| Q | Open Sensor QA |

Space retains native activation on buttons, links, form controls, summaries,
and contenteditable elements.
