# Hugging Face Destination

Hugging Face Hub is visible as an explicit but unavailable network destination
in the source build.

The application does not request a token, store a credential, create a
repository, or upload files. Selecting Hugging Face Hub displays a blocked
status and disables the confirmation action.

To produce evidence locally, choose `Local disk`, `Embodiment card`, or
`Failure gallery`. Those destinations download a real ZIP archive with
deterministic files and SHA-256 checksums.
